# Pisme Arbiter Search

Search engine for court arbitrators that connects best fitting arbitraitors to court cases. Winner of special mention at Hack4Law hackathon

Features:

-   Multi-step User registration
-   Axios-based search

Technologies:

-   Axios
-   Chakra UI
-   React
-   Zustand
-   JWT
